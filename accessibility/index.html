Accessibility

This website is partially compliant with WCAG 2.1, level AA. Efforts to improve accessibility are ongoing. Current known issues include insufficient text color contrast, missing or incomplete labels for images and tables, and missing ARIA roles in some navigation elements.

Barriers encountered while using the site can be reported by sending feedback, a short description of the issue, and the affected URLs to andreas.bernkop@uibk.ac.at

If the response is not satisfactory, a complaint may be submitted to the Austrian Research Promotion Agency (FFG). Complaints can be filed online (form in German). The FFG reviews whether the matter concerns non-compliance with the Web Accessibility Act (WZG). If the complaint is justified, the FFG recommends further action to the responsible entities and proposes measures to resolve the issue.
